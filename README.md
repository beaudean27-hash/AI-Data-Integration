# Mobile Wifi Comparison

This repository contains a single-page static site comparing 5G mobile hotspot devices and data plans. It includes a short audio narration.

How to view locally

1. Open `index.html` in a browser (double-click or drag into browser).
2. Click the audio player labeled "Please Push Play" to hear the narration.

How to publish to GitHub Pages

1. Create a new repository on GitHub named `Mobile-Wifi-Comparison`.
2. Upload the contents of this folder (or push via Git) to the repo's `main` branch.
3. Enable GitHub Pages in the repo Settings â†’ Pages using the `main` branch and the `/(root)` folder.

Notes

- Keep the `audio/` folder next to `index.html` so relative paths work.
- If you prefer a shorter audio filename, rename it and update `index.html` accordingly.
